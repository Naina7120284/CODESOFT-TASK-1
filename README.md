# JobBoard - Full Stack MERN Application

###  Key Features
* **Monorepo Architecture**: Unified backend and frontend management.
* **Live Integration**: Re-connected Render backend with Render frontend via environment variables.
* **Responsive Design**: Mobile-optimized UI for all dashboard stats.

### Tech Stack
* **Frontend**: React.js, CSS
* **Backend**: Node.js, Express.js
* **Database**: MongoDB Atlas
